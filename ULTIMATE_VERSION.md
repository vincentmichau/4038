## 🔧 Installation One-Shot

### Installation Automatisée
```bash
# Clone du projet
git clone https://github.com/nightguards/system.git
cd nightguards

# Installation one-shot (tout inclus)
chmod +x install.sh
./install.sh

# Démarrage automatique
./start.sh
```

### Installation Docker
```bash
# Build et déploiement Docker
docker-compose up -d

# Accès à l'application
http://localhost:3000
```

### Installation Manuelle
```bash
# Installation dépendances
npm run install:all

# Configuration base de données
npm run db:setup

# Configuration sécurité
npm run security:setup

# Build applications
npm run build:all

# Démarrage
npm start
```
